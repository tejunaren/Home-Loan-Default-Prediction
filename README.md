<h3 style="color:#1F618D; font-family:'Book Antiqua'; font-weight:bold;">
📌 Project Overview
</h3>

<p style="font-size:16px; font-family:'Cambria'; line-height:1.8;">
<b>This project focuses on predicting loan default risk using Machine Learning techniques.</b> 
The dataset contains multiple customer financial and credit-related features such as 
<b>income details, credit amount, previous loan history, installment payments, 
bureau records, and customer demographic information.</b>
</p>

<p style="font-size:16px; font-family:'Cambria'; line-height:1.8;">
The primary objective of this project is to:
</p>

<ul style="font-size:16px; font-family:'Cambria'; line-height:1.8;">
<li><b>Analyze customer loan behavior</b></li>
<li><b>Preprocess banking datasets</b></li>
<li><b>Perform feature engineering</b></li>
<li><b>Identify high-risk loan applicants</b></li>
<li><b>Build predictive classification models</b></li>
</ul>

<p style="font-size:16px; font-family:'Cambria'; line-height:1.8;">
Several machine learning algorithms including:
</p>

<ul style="font-size:16px; font-family:'Cambria'; line-height:1.8;">
<li><b>Logistic Regression</b></li>
<li><b>Decision Tree Classifier</b></li>
<li><b>Random Forest Classifier</b></li>
<li><b>Gradient Boosting Classifier</b></li>
</ul>

<p style="font-size:16px; font-family:'Cambria'; line-height:1.8;">
were trained and evaluated using performance metrics such as:
</p>

<ul style="font-size:16px; font-family:'Cambria'; line-height:1.8;">
<li><b>Accuracy Score</b></li>
<li><b>Confusion Matrix</b></li>
<li><b>Classification Report</b></li>
<li><b>ROC Curve</b></li>
<li><b>ROC-AUC Score</b></li>
</ul>

<p style="font-size:16px; font-family:'Cambria'; line-height:1.8;">
Finally, the <b>best-performing model</b> was selected based on prediction performance, 
generalization ability, and classification accuracy.
</p>

<p style="font-size:16px; font-family:'Cambria';">
<b>Insight:</b> Proper preprocessing, feature engineering, and ensemble machine learning models significantly improved loan default prediction accuracy.
</p>
